About This Web App



This web application is a test object-detection mechanism, which was created for the Kchat corporation to help identify images shared on the Kchat platform.
It has many demos available for the user to check the capability of the object-detection mechanism.
This web app also has interactive and interesting UI for the user to enjoy.
It also contains many beautiful graphics and images which are user-friendly.
